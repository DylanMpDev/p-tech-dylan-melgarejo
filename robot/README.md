




#Robot cleaner Technical Test:
#Author: Jeffer Dylan Melgarejo

This project is a backend technical test implemented in Java 17 using Spring Boot.

The goal of the application is to simulate the movement of an autonomous cleaning robot on a grid-based factory floor, following a set of simple instructions.

## Technology
- Java 17
- Spring Boot
- Maven

## How to run
```bash
./mvnw spring-boot:run
